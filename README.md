# Routine Assistant

A simple productivity web application that combines a **to-do list**, **notes system**, and **clicker game** in one interface.

All data is stored locally in the browser using **Local Storage**, which means no backend or database is required.

---

## Features

### To-Do List
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Automatically save tasks in Local Storage

### Notes
- Create and edit notes
- Instant saving
- Persistent browser storage

### Clicker Game
- Increase counter by clicking on button
- Buy automations
- Earn credits

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Local Storage API
- ChatGPT for explanation of complex tasks.
